🧮 Основные алгоритмы
1️⃣ Z-буфер (без сортировки!)
Подход: для каждого пикселя интерполируем Z через барицентрические координаты и сравниваем с буфером. Не требует предварительной сортировки граней — правильный порядок достигается автоматически!

python
# Для каждого пикселя в треугольнике:
z_interpolated = u*z1 + v*z2 + w*z3  # барицентрическая интерполяция
if z_interpolated < z_buffer[y][x]:
    z_buffer[y][x] = z_interpolated
    screen.set_at((x, y), color)
    
2️⃣ Барицентрические координаты
Все пиксели внутри треугольника вычисляются через веса (u, v, w):
Интерполяция Z для скрытых поверхностей
Интерполяция цвета и нормалей
Естественная проверка "внутри ли точка" в одну строку

3️⃣ Lambert Shading
intensity = ambient + max(0, normal · light_direction) * (1 - ambient)
Диффузное освещение через скалярное произведение — реалистично и быстро.

4️⃣ Backface Culling
Отсекаем грани, повёрнутые от камеры через проверку normal · to_camera > 0. Экономит 50% времени рендеринга!

5️⃣ Матричная математика
Полная реализация без NumPy:
Умножение матриц 4×4 (тройной цикл)
Матрицы вращения (X, Y, Z оси)
Перспективная проекция (NDC → Screen)

6️⃣ Алгоритм Брезенхэма
Рисование линий только целыми числами — для wireframe режима без ошибок округления.

Все генерируются параметрически через UV-разбиение:

Остальные: ParametricSurface с произвольной функцией (u, v) → (x, y, z)

Сравнение подходов:
